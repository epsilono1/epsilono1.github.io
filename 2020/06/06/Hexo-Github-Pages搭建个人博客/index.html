    Hexo+Github Pages搭建个人博客

    系统：win7 64位 我的博客：Epsilono1

    一、github建立blog仓库

    1. 创建新仓库： “用户名”.github.io

    2. 配置SSH Keys

      注：可能你原来注册github时弄过ssh keys，可以先本地搜索下id_rsa.pub文件，如果已经有了，说明已经配置过，就可略过整个配置过程。

      1> 生成SSH Keys

      ssh-keygen -t rsa -C "github邮箱"

      一路回车。配置完会有id_rsa.pub文件。

      2> 复制SSH Keys

      记事本打开 id_rsa.pub文件，复制里面的 SSH Keys

      3> 配置SHH Keys

      github头像–>settings–>SSH and GPG keys–>New SSH key—>Title(随便填)，Key(粘贴)–>Add

    二、安装配置Hexo

    安装 hexo 前，请先安装 Node.js 和 Git 。hexo安装如下:

    1. hexo安装：

      1>本地找个地儿建一个文件夹（../Hexo/blog），用于存放blog。

      2>在此文件夹（blog）下，右键–>Git Bush Here，打开命令窗口。

      由于npm官方安装源在国外，修改一下npm安装源为淘宝npm镜像：

      npm config set registry https://registry.npm.taobao.org

      完后开始安装Hexo：

      npm install -g hexo-cli

      完后运行 hexo命令，检测是否安装成功。

    2. blog初始化：

      在blog文件夹下，运行命令

      hexo init

      完后，打开_config.yml修改配置信息：

      url: http://yoursite.com ##把这个改成你的github地址（例如：http://myKepler.github.io）

deploy:
    type: git  ##注意空格
    repo: git@github.com:myKepler/myKepler.github.io.git ##仓库地址 
    branch: master

    三、第一篇文章

    1. 新建文章

      hexo new "文章标题"

      该文章是md格式，位置在source/_posts中。

    2. 本地预览

      hexo s

      如果正常执行，会有个 http://localhost:4000 ，浏览器中输入该网址即可预览文章。

    3. 发布博客

      hexo g #生成静态文件存于public

      hexo d #部署到站点

      执行两命令后即可成功发布。可以在个人站点（用户.github.io）查看。

    四、主题更改

    默认主题是landscape，不喜欢可以自己更改一个，我用的是 3-hexo 主题（三段式），你可以用这个，也可以去 hexo主题官网 按需下载，github主题下载命令(blog下执行)：

    git clone https://github.com/yelog/hexo-theme-3-hexo.git

    下载到本地blog目录下，根据主题开发者提供的教程更改配置即可。
